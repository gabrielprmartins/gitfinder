## :gear: Installation
```bash
  # Clone the repositorie
  $ git clone https://github.com/gabriuda/gitfinder

  # Get in
  $ cd gitfinder

  # Install as dependencies
  $ npm install

  # Developer mode
  $ npm run dev

  # Use Live Server to view it on your machine
```
